## Image Guidelines:
- Use JPG for photos, PNG for logos
- Recommended size: 1200x800px for banners, 400x300px for gallery
- Keep file sizes under 2MB for faster loading
- Use descriptive filenames as shown above

The website will automatically detect and use these images when you upload them.
